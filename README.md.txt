# Superposition Demo (Single-Qubit Unitary)

Interactive Jupyter notebook with sliders for \( \theta,\phi,\lambda \); shows pre-built and custom unitary, eigenvalues, eigenvectors, and a histogram with \(p_{\mathrm{theo}}\), \(p_{\mathrm{samp}}\), and counts.

**Version:** v1.0  
**Date:** 2025-10-16

## Requirements
Python 		>= 3.12.2
numpy      	>= 1.26.4
matplotlib 	>= 3.10.0
ipywidgets 	>= 8.1.5

### Run
Open in **JupyterLab** and run all cells:
1) `jupyter lab`
2) Open `sim_superposition.ipynb`
3) Run the notebook
